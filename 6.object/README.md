# 객체

## 이 폴더의 목적

객체의 핵심 특성(프로퍼티 접근 방식, 참조 공유, 얕은 복사)을
코드로 확인한다.

## 구성

- **1.notation.js**
  - 점 표기법 vs 대괄호 표기법
  - 동적 키(런타임에 결정되는 키) 접근은 `[]`를 사용한다.

- **2.reference.js**
  - 객체는 할당 시 참조가 복사되어 같은 객체를 공유할 수 있다.

- **3.shallow-copy.js**
  - spread로 복사해도 얕은 복사이며, 중첩 객체는 여전히 공유될 수 있다.

## 핵심 포인트

- `obj.key`는 정적 키 접근, `obj[key]`는 동적 키 접근에 사용한다.
- 객체는 참조 타입이라 의도치 않은 공유/변경이 발생할 수 있다.
- 얕은 복사는 최상위만 새로 만들기 때문에, 중첩 객체는 공유될 수 있다.

## 한 문장 정리

객체는 참조를 다루기 때문에, 접근/복사/비교에서 “공유”가 발생할 수 있음을 항상 의식해야 한다.
