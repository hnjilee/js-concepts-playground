# 배열

## 이 폴더의 목적

배열의 핵심 특성(참조 타입)과 자주 쓰는 기본 메서드(map/filter/forEach)를
코드로 확인한다.

## 구성

- **1.reference.js**
  - 배열은 참조 타입이라 복사 시 같은 배열을 공유할 수 있다.
  - 원본을 직접 수정하지 않고 새 배열을 만드는 불변 업데이트 방식(`...spread`)을 사용한다.

- **2.methods.js**
  - `map`: 변환(새 배열 반환)
  - `filter`: 조건에 맞는 요소만 필터링(새 배열 반환)
  - `forEach`: 각 요소에 대해 실행(반환값 없음, 주로 side effect 목적)

## 핵심 포인트

- 배열도 객체이므로 참조가 공유될 수 있다.
- 불변성을 지키기 위해서 원본 배열을 직접 수정하지 말고 새 배열을 만들어 업데이트한다.
- `map/filter`는 “새 배열을 만드는” 메서드, `forEach`는 “실행” 목적이다.

## 한 문장 정리

배열은 순서 있는 참조 타입이므로, 변형은 불변 방식으로 처리하는 것이 안전하다.
