# 연산자 (논리 연산자와 단락 평가 위주로)

## 코드 작성 목적

- 연산자의 동작을 확인하고,
- 특히 논리 연산자의 단락 평가를 직접 확인하기 위해

## 확인한 동작

- `+` 연산자는 문자열이 포함되면 문자열 결합으로 동작한다
- 일치 연산자는 값과 타입을 함께 비교한다
- 논리 연산자는 Boolean이 아니라 **마지막으로 평가된 값을 반환**한다
- 이 특성으로 인해 단락 평가가 발생한다

## 단락 평가의 활용 예시

- 조건이 truthy일 때만 함수 호출 (`&&`)
- falsy 값을 대체하는 기본값 설정 (`||`)

## 한 문장 정리

논리 연산자는 Boolean을 반환하는 연산자가 아니라,
평가 결과 값을 반환하는 연산자이기 때문에 단락 평가 패턴이 가능하다.
