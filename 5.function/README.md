# 함수

## 이 폴더의 목적

함수의 다음과 같은 핵심 특성을 코드로 확인한다.

- 실행 시점을 제어하고
- 외부 값을 받아 처리하고
- 결과를 반환하며
- 실행을 다른 함수에 위임할 수 있는 구조

## 구성

- **1.declaration-vs-call.js**
  - 함수는 호출 전에는 실행되지 않는다. (선언 ≠ 호출)

- **2.parameter.js**
  - 매개변수를 통해 외부 값을 받아 함수를 재사용할 수 있다.

- **3.return.js**
  - `return`으로 결과값을 반환함으로써 외부에서 사용할 수 있다.

- **4.callback.js**
  - 함수를 다른 함수에 인자로 전달하여 실행을 위임할 수 있다.

## 핵심 포인트

- 함수는 호출될 때 실행된다.
- 매개변수는 외부 값을 함수 내부로 전달받는 통로이다.
- return는 함수 실행의 결과값을 반환하고 실행을 종료한다.
- 함수는 값으로 전달될 수 있으며, 이를 통해 실행을 위임해 실행 시점을 제어할 수 있다.

## 한 문장 정리

함수는 코드를 재사용하기 위한 구조이면서,
실행 시점과 코드 흐름을 제어하는 단위다.
