# 제어문

## 이 폴더의 목적

자바스크립트에서 코드의 실행 흐름을 어떻게 제어하는지 코드를 통해 확인한다.

## 구성

- **1.conditional.js**
  - 조건식 내 논리 연산자는 마지막 평가 값을 반환한다.
  - 반환된 값은 truthy/falsy로 평가된다.

- **2.loop.js**
  - 반복문은 조건이 만족되는 동안 실행된다.
  - 종료 조건이 없거나 유효하지 않으면 무한 루프에 빠질 수 있다.

- **3.break-and-continue.js**
  - `break`: 반복문 전체를 조기 종료
  - `continue`: 현재 반복만 건너뛰고 다음 반복으로 넘어감

## 핵심 포인트

- 조건식에는 Boolean만 들어가는 것이 아니라 어떤 값이든 들어갈 수 있다.
- 그 값이 Boolean으로 평가되어 실행 여부가 결정된다.
- 반복문은 조건이 만족되는 동안 실행되므로, 무한 루프를 방지하기 위해 유의미한 종료 조건 설정이 중요하다.
- `break`와 `continue`는 반복 흐름을 중간에 제어하기 위한 도구다.

## 한 문장 정리

제어문은 조건과 반복으로 실행 흐름을 제어한다.
