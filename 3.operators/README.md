# 연산자 (논리 연산자와 단락 평가 위주로)

## 이 폴더의 목적

논리 연산자가 Boolean이 아니라 마지막 평가 값을 반환함을 코드로 확인한다.

## 확인한 핵심 동작

- `&&`는 앞의 피연산자가 truthy이면 뒤도 평가하고, falsy이면 평가하지 않는다.
- `||`는 앞의 피연산자가 falsy이면 뒤도 평가하고, truthy이면 평가하지 않는다.
- 논리 연산자는 마지막으로 평가된 값을 반환한다.

## 코드로 확인한 예시

- 조건이 true일 때만 함수 실행 (`&&`)
- falsy 값을 대체하는 기본값 설정 (`||`)

## 한 문장 정리

논리 연산자는 Boolean을 반환하는 연산자가 아니라,
평가 결과 값을 반환하는 연산자이기 때문에 단락 평가 패턴이 가능하다.
